# Agent Instructions

This file contains project-specific instructions for agents working in this repository.

Follow all development, style, testing, and versioning guidelines in [Contributing.md](Contributing.md).

## Agent-Specific Notes

- Always place test output, dumps, and generated files in the `.work/output/` directory (gitignored — do not commit).
- Prefer PowerShell scripts for quick prototyping; use a temporary console app under `.work/` for richer exploration.
- Do not commit one-off test harnesses into the main project.
- When the body of work is complete, do a review of all the code and the current change set and see if anything can be refactored or improved.
